Proyecto personal: ideado y desarrollado de principio a fin sin cliente real.

Objetivo

Diseñar una web responsive para el sello discográfico Panorama que reflejara su identidad visual y facilitara el descubrimiento de música y artistas.

Proceso

Investigación

Entrevistas a seguidores del sello para entender cómo descubren música nueva y qué esperan de una web de un sello independiente.

Análisis de otras webs de sellos y tiendas de merchandising para identificar patrones de navegación y oportunidades de diferenciación visual.
